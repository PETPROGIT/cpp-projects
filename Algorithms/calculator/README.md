# Калькулятор арифметических выражений

## Назначение

Модуль предназначен для вычисления значений арифметических выражений, заданных в виде строки. Реализация основана на алгоритме Shunting Yard, разработанном Э. Дейкстрой.

## Функциональные возможности

- Поддержка бинарных арифметических операций:
  - сложение (`+`)
  - вычитание (`-`)
  - умножение (`*`)
  - деление (`/`)
- Поддержка группирующих символов: круглые скобки (`(`, `)`)
- Обработка вещественных чисел с плавающей точкой

## Ограничения

Модуль **не поддерживает** унарный минус. Выражения, начинающиеся с символа `-` или содержащие унарный минус внутри, признаются синтаксически некорректными.

## Обработка ошибок

В процессе вычисления могут быть сгенерированы следующие исключения:

| Тип исключения | Условие возникновения |
|----------------|----------------------|
| `SyntaxError` | Синтаксическая ошибка во входном выражении: непарные скобки, недопустимые символы, некорректный формат числа, отсутствие операнда, некорректная расстановка операторов |
| `DivisionByZero` | Попытка выполнения операции деления с нулевым делителем |

Оба типа исключений наследуют `std::runtime_error` и содержат текстовое описание ошибки.
